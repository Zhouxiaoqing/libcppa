virtual void cppa::local_actor::quit ( std::uint32_t  reason = exit_reason::normal)
virtual

Finishes execution of this actor after any currently running message handler is done.

This member function clear the behavior stack of the running actor and invokes on_exit(). The actors does not finish execution if the implementation of on_exit() sets a new behavior. When setting a new behavior in on_exit(), one has to make sure to not produce an infinite recursion.

If on_exit() did not set a new behavior, the actor sends an exit message to all of its linked actors, sets its state to exited and finishes execution.

Parameters
reasonExit reason that will be send to linked actors and monitors. Can be queried using planned_exit_reason(), e.g., from inside on_exit().
Note
Throws actor_exited to unwind the stack when called in context-switching or thread-based actors.
Warning
This member function throws imeediately in thread-based actors that do not use the behavior stack, i.e., actors that use blocking API calls such as receive().

Reimplemented in cppa::io::broker.
