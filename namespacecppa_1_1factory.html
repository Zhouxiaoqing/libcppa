template<typename InitFun >
detail::ebaf_from_functor<InitFun, void (*)()>::type cppa::factory::event_based ( InitFun  init)

Returns a factory for event-based actors using fun as implementation for init().

fun must take pointer arguments only. The factory creates an event-based actor implementation with member variables according to the functor's signature, as shown in the example below.

* auto f = factory::event_based([](int* a, int* b) { ... });
* auto actor1 = f.spawn();
* auto actor2 = f.spawn(1);
* auto actor3 = f.spawn(1, 2);
*

The arguments a and b will point to int member variables of the actor. All member variables are initialized using the default constructor unless an initial value is passed to spawn.
